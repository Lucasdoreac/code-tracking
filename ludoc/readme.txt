Um tema premium WordPress com suporte completo ao Elementor, modo escuro integrado e recursos multilíngues.

== Description ==
O Tema Ludoc é a solução completa para agências e profissionais de desenvolvimento web que buscam criar sites modernos e profissionais. Com suporte total ao Elementor, modo escuro nativo e recursos multilíngues, o Ludoc oferece tudo que você precisa para criar projetos excepcionais.

== Installation ==
1. Acesse o painel WordPress
2. Navegue até Aparência > Temas
3. Clique em "Adicionar Novo"
4. Clique em "Fazer Upload do Tema"
5. Selecione o arquivo ludoc.zip
6. Clique em "Instalar Agora"
7. Após a instalação, clique em "Ativar"

== Frequently Asked Questions ==
= O tema é compatível com o Elementor? =
Sim, o tema foi desenvolvido com suporte total ao Elementor e Elementor Pro.

= Como ativar o modo escuro? =
O modo escuro pode ser ativado através do botão no cabeçalho do site.

= O tema suporta múltiplos idiomas? =
Sim, o tema é compatível com WPML e Polylang.
